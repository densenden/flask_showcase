Route Overview

/?name=John

Displays the homepage and optionally greets the user based on the 'name' query parameter in the URL. Defaults to 'No Name' if not provided.

/greet/Alice

Takes a 'name' from the URL and displays a greeting with the capitalized version of the name.

/form

Renders a form page where the user can submit data (e.g., updating user information).

/all-users

Displays a list of all users (name, age, and country) from the 'users' dictionary.

/all-users/table

Displays a table representation of all users from the 'users' dictionary.

/update-country

Allows updating a user's country. If the user exists, it updates the country; otherwise, it adds the user with the provided country, without age. Redirects to the 'all_users' route after updating.
